## sorting visualization tool 

### Covered Algorithms: 

| Algorithm      | Time complexity  | 
| -------------- |:----------------:|
| Bubble sort    | O(n<sup>2</sup>) |
| Selection sort | O(n<sup>2</sup>) |
| Insertion sort | O(n<sup>2</sup>) |
| Merge sort     | O(n*log<sub>2</sub>n)|
| Quick sort     |~O(n*log<sub>2</sub>n)|
